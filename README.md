```
This cli app recives url as an argument, downloads html page and
modify all the references on it to the local resources.
```

## Install:
```
make install
```
## Setup as CLI app:
```
npm install -g pgloader
```
## Setup as dependence:
```
npm install --save pgloader
```

## Usage:
```
 pgloader --output [path] <URL>
```
