```
This cli app downloads and save html page by given URL and
modify all the references to the local resources.
```

## Install:
```
make install
```
## Setup as CLI app:
```
npm install -g pgloader
```
## Setup as dependence:
```
npm install --save pgloader
```
## Usage:
```
 page-loader --output [path] <URL>
```
